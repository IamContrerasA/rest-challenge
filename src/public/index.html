Iam Rest Challenge

The endpoints for the app are:

Auth

Please use imnsonia or postman

First signup with your email name@ravn.co, then check your email and you have your email token, verify your account and you can use the rest of endpoints. Use the beared token gived in insomnia for example

POST: https://iamrestchallenge.herokuapp.com/signup
POST: https://iamrestchallenge.herokuapp.com/signin
POST: https://iamrestchallenge.herokuapp.com/verify-account/[token]
POST: https://iamrestchallenge.herokuapp.com/signout

App

all end points show in the image

PUT: https://iamrestchallenge.herokuapp.com/posts/1/comments/1/liked ------ like a comment
PUT: https://iamrestchallenge.herokuapp.com/posts/1/comments/1/publish ------ publish a comment because is draft in first place
DEL: https://iamrestchallenge.herokuapp.com/posts/1/comments/1 ------ delete a comment
PUT: https://iamrestchallenge.herokuapp.com/posts/1/comments/1 ------ update a comment
POST: https://iamrestchallenge.herokuapp.com/posts/1/comments ------ create a comment for example{ "content": "post7 comment 2" }
GET: https://iamrestchallenge.herokuapp.com/posts/1/comments/1 ------ get a comment with id
GET: https://iamrestchallenge.herokuapp.com/posts/1/comments ------ get all comments
------------------------------------------------------------------------------------------------------------------------------------------------------------------
PUT: https://iamrestchallenge.herokuapp.com/posts/1/liked ------ like a post
PUT: https://iamrestchallenge.herokuapp.com/posts/1/publish ------ publish a post because is draft in first place
DEL: https://iamrestchallenge.herokuapp.com/posts/1 ------ delete a post
PUT: https://iamrestchallenge.herokuapp.com/posts/ ------ update a post
POST: https://iamrestchallenge.herokuapp.com/posts ------ create a post for example { "title": "post title7", "content": "post content7" }
GET: https://iamrestchallenge.herokuapp.com/posts/1 ------ get post 1
GET: https://iamrestchallenge.herokuapp.com/posts ------ get all posts
------------------------------------------------------------------------------------------------------------------------------------------------------------------
PUT: https://iamrestchallenge.herokuapp.com/accounts/1/publicEmail ------ change isEmailPublic to true
PUT: https://iamrestchallenge.herokuapp.com/accounts/1/publicName ------ change isNamePublic to true
DEL: https://iamrestchallenge.herokuapp.com/accounts/1 ------ delete a user
PUT: https://iamrestchallenge.herokuapp.com/accounts/1 ------ Update a user. For example { "firstname": "Primer noombre", "surname": "Apellido" }
GET: https://iamrestchallenge.herokuapp.com/accounts/1 ------ Get user 1
GET: https://iamrestchallenge.herokuapp.com/accounts ------ Get all users
